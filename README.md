# near-kit-tool-box-rs
🦀 near kit rs tool box, functions and more for specific near interactions

ℹ️ primarly for personal use
<br/>
ℹ️ the lib contains reusable functions the bins are examples. 

---

**CARGO COMMANDS**

```sh
cargo run
cargo check
# cargo check --target wasm32-unknown-unknown
# cargo test
cargo clean
cargo fmt

# docs
cargo doc --no-deps
cargo doc --open

# mdbook
mdbook serve --open
```

**dependencies**
- near-kit-rs (```cargo add near-kit```)
